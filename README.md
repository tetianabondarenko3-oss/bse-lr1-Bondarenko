# Назва проєкту

Touch Up Smart
це веб-сервіс, призначений для швидкого підвищення рівня ерудиції та підготовки до спілкування перед різними подіями (зустрічі, побачення, ділові розмови тощо). Система дозволяє користувачеві обирати категорію події та отримувати підбірку тем або фактів для розмови, що допомагає уникнути незручних пауз та зробити спілкування більш впевненим і цікавим.
